Curso de HTML5 com CSS3

HTML5, sigla para Hypertext Markup Language, é uma linguagem de marcação de hipertexto para apresentar e estruturar o conteúdo na web. O HTML5 é a versão mais recente da linguagem de programação conhecida como HTML (Hyper Text Markup Language), sigla em Inglês para Linguagem de Marcação de Hipertexto. Essa linguagem usa um texto codificado para criar e organizar conteúdo online.
